# Telegram Guardian Enterprise

✅ Live /status reporting bot  
✅ Inline keyboard for service URLs  
✅ Configurable services via services.json  
✅ Polling loop for 24/7 operation  
✅ Safe Git setup: sensitive info excluded

## Setup
1. Update ot\config.ps1 with your bot token and chat ID.
2. Run pwsh .\bot\telegram_guardian.ps1
3. Bot will respond to /start and /status in Telegram.
